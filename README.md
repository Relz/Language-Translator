# Language Translator
## Пример ввода леволинейной грамматики:<br/>
```
S > Sa | Aa | Hd
A > Aa | Bb
B > Bb | Hc
```

## Пример ввода праволинейной грамматики:<br/>
```
S > aS | aB
B > bB | bC
C > bC | cH
```

В результате работы программа создаёт исходный и детерминированный автомат в виде изображений
